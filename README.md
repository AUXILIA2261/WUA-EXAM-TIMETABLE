# \# WUA Automated Examination Timetable Scheduling System

# 

# \## Problem Statement

# At Women's University, exam timetables are created manually,

# leading to clashing exams, double booking of time slots,

# and some courses being omitted.

# 

# \## Proposed Solution

# A software system that automatically generates conflict-free

# exam timetables for all courses, rooms, and time slots.

# 

# \## Objectives

# \- Eliminate exam clashes

# \- Ensure all courses are scheduled

# \- Reduce human errors

# \- Improve efficiency
